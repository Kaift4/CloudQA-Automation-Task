# CloudQA Selenium Automation Task

This project contains automated UI tests in **C# + Selenium** for the CloudQA Automation Practice Form.

## Features
- Page Object Model (POM)
- Stable, resilient locators that do not break with HTML changes
- Three automated tests:
  - First Name field
  - Gender selection
  - Submit button

## Requirements
- .NET SDK
- Selenium WebDriver
- ChromeDriver
- NUnit

## Running Tests
1. Install dependencies:
   ```
   dotnet restore
   ```
2. Run the tests:
   ```
   dotnet test
   ```
